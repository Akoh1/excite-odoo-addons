Odoo Rest API

A REST API is an application program interface (API) that uses HTTP requests to GET, PUT, POST and DELETE data.

In this module, we create an Odoo REST API endpoints which touch the Database records and Schema in JSON and XML format.

Endpoints Allow

  • Fetch a record from Database.
  • Create a record in Database
  • Update a record in Database
  • Delete a record in Database
  • Search records in Database
  • Get the Schema of a Table.
  • Trigger your odoo actions.
  • Access rights message in Endpoint response.
  • Quick and Easy Installation.

  • Admin Control
    • Create API key to access Endpoints
    • Seperate API key for end point users
    • Give different access right to multiple API key
